### 사용자 요청 및 아이디어 기록

#### 1. 인풋/아이디어 테두리 색상 통일
- **요청:** `IdeaInput` 컴포넌트의 테두리 색상을 모달 및 홈 화면 보더와 동일한 `#6BB9FA` 색상으로 통일해달라. (`IdeaCard`는 이미 `.card` 클래스에 의해 통일된 것으로 추정됨.)
- **실행 방안:** `IdeaInput.tsx` 파일을 수정하여 `<Form.Control>` 요소의 `style` prop에 `borderColor: '#6BB9FA', borderWidth: '2px', borderStyle: 'solid'` 등을 추가.

#### 2. 모달 창 정보 표시 문제
- **요청:** 아이디어를 클릭했을 때 `IdeaDetailModal`의 정보가 제대로 표시되지 않는 문제가 있다.
- **조치 방안:**
    - `functions/api/idea-detail.ts` 파일의 OpenAI 프롬프트가 상세 정보를 구조화된 형태로 잘 요청하고 있는지 재확인.
    - `functions/_utils/openai.ts` 파일의 `parseAIResponse` 함수 (상세 정보용)가 AI 응답의 구조를 정확히 파악하여 `details` 배열에 담아 반환하고 있는지 검토 및 수정.
    - `IdeaDetailModal.tsx`에서 `details` prop을 렌더링하는 로직이 파싱된 데이터를 올바르게 출력하는지 확인. 특히 여러 줄의 정보가 구조화되어 나타나야 하므로, 단순히 `<p>` 태그로만 묶는 것이 아닌, 헤딩과 목록 등을 인식하여 적절한 HTML 태그로 렌더링하는 로직 추가 검토.

#### 3. 모달 UI 변경
- **요청:** 모달 창의 'X' 버튼을 없애고, '닫기' 버튼의 테두리 색상을 모달 보더 색상 (`#6BB9FA`)과 동일하게 변경해달라.
- **조치 방안:**
    - **'X' 버튼 제거:** `IdeaDetailModal.tsx`와 `AuthForm.tsx`에서 `Modal.Header` 컴포넌트의 `closeButton` prop을 `false`로 설정.
    - **'닫기' 버튼 색상:** `IdeaDetailModal.tsx` 파일 내의 `Modal.Footer`에 있는 '닫기' `Button` 컴포넌트의 `style` prop에 `borderColor: '#6BB9FA', color: '#6BB9FA'` (투명 배경) 또는 `backgroundColor: '#6BB9FA', color: 'white'` (채워진 배경) 등을 적용. 호버 효과도 고려.

#### 4. 본 웹사이트에 대한 탭 기능 아이디어
- **아이디어 1: "아이디어 탐색" / "찜한 아이디어" 탭**
    - **설명:** 현재 별도의 섹션으로 분리된 아이디어 생성 결과 목록과 사용자가 찜한 아이디어 목록을 하나의 탭 인터페이스로 통합하여, 사용자가 상단 탭 클릭만으로 두 목록 간을 쉽게 전환하며 볼 수 있도록 UX 개선.
- **아이디어 2: "생성" / "트렌드" 탭**
    - **설명:** 사용자가 새로운 아이디어를 생성하는 메인 기능 탭과 함께, 현재 웹사이트 내에서 인기 있는 아이디어 키워드, 최신 트렌드 데이터(예: 검색량 증가 키워드)를 시각화하여 보여주는 "트렌드" 탭을 추가. 이를 통해 사용자가 아이디어 발상에 영감을 얻을 수 있도록 지원.
- **아이디어 3: "내 아이디어" / "공유 아이디어" 탭 (향후 소셜 기능 추가 시)**
    - **설명:** 사용자가 직접 생성하고 관리하는 아이디어 목록을 보여주는 탭과, 다른 사용자들이 공유하고 공개한 아이디어들을 탐색할 수 있는 "커뮤니티" 또는 "공유 아이디어" 탭을 구분하여 제공.

#### 5. 우리 웹사이트(AI 아이디어 추천기)에 대한 추가 기능 아이디어 (총 4가지)
- **아이디어 1: 아이디어 필터링 및 정렬 기능**
    - **설명:** 생성된 아이디어 목록이나 찜한 아이디어 목록에서 사용자가 특정 키워드, 경쟁 강도 (낮음/보통/높음), 유망성 이유 등을 기준으로 필터링하고, 최신순/오래된순/인기순 등으로 정렬할 수 있는 기능을 추가합니다. 이는 사용자가 방대한 아이디어 중에서 원하는 정보를 효율적으로 찾을 수 있도록 돕습니다.
- **아이디어 2: 아이디어 상세 분석 보고서 생성**
    - **설명:** 특정 아이디어의 상세 모달에서 "보고서 생성" 버튼을 추가하여, 해당 아이디어에 대한 AI 분석 결과(틈새시장, 아이디어, 유망성, 경쟁 강도 등)를 PDF 또는 이미지 파일 형태로 다운로드할 수 있는 기능을 제공합니다. 이는 사용자가 아이디어를 다른 사람과 공유하거나 오프라인에서 검토할 때 유용합니다.
- **아이디어 3: 사용자별 아이디어 선호도 학습 및 맞춤 추천**
    - **설명:** 사용자가 찜한 아이디어, 자주 검색하는 키워드, 상세 정보를 조회한 아이디어 등을 AI가 학습하여, 다음 아이디어 생성 시 사용자에게 더 적합한 테마나 스타일의 아이디어를 우선적으로 제시하는 맞춤 추천 기능을 구현합니다. 이는 개인화된 사용자 경험을 제공하여 서비스 만족도를 높입니다.
- **아이디어 4: 아이디어 발전 및 협업 공간**
    - **설명:** 찜한 아이디어를 단순 저장하는 것을 넘어, 각 아이디어에 대한 사용자의 메모, 추가적인 생각, 할 일 목록 등을 기록하고 관리할 수 있는 개인 공간을 제공합니다. (선택적으로) 다른 사용자들과 함께 아이디어를 발전시킬 수 있는 협업 기능(댓글, 수정 제안 등)을 추가하여 아이디어 구체화 과정을 지원합니다.
